# MusicBaBa

Simple MP3 player made in python with pygame's mixer and tkinter as GUI  (*old project of 20 Oct'19*)


### Installation 

- requirement of __pygame__ *ver1.9.6*
   
```
pip install pygame==1.9.6
```
